# Amazon E-Commerce Analytics SQL Project

## 📖 Overview
This project is a SQL-based Amazon e-commerce database designed to manage and analyze customer, product, order, and shipping data. It demonstrates relational database design and business data analysis using SQL queries.

## 🗄️ Database Tables
- Customers
- Categories
- Products
- Orders
- Order_Details
- Shipping

## 🛠️ SQL Concepts Used
- Database Creation
- Table Creation
- Primary & Foreign Keys
- Data Insertion
- Joins
- Aggregate Functions (SUM, AVG)
- Subqueries
- Views

## 📊 Analysis Performed
- Total Sales Analysis
- Average Profit Calculation
- Customer Purchase Analysis
- Product Performance Analysis
- Sales Summary Reporting

## 💻 Technologies Used
- SQL
- MySQL
